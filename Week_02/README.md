### Sorted 数据结构优化
###### 原型添加take与give方法
- 在数据结构中，每次找出与终点最近的之，take到数据外
- 方法：遍历循环，寻找最小值，并删除

```
function distance(point) {
    return (end[0] - point[0]) ** 2 + (end[1] - point[1]) ** 2
}
```

- 效果：很多离终点远的点，不在搜索范围内
